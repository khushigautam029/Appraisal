# 📊 Appraisal Management System

A web-based **Employee Appraisal Management System** developed to automate and streamline the employee performance evaluation process within an organization.

The system allows **Employees, Managers, and HR/Admin** to manage the complete appraisal lifecycle digitally — from self-evaluation to final appraisal decisions and reporting.

---

## 📌 Project Overview

Traditional employee appraisal methods often involve manual paperwork, spreadsheets, and lengthy approval workflows, resulting in delays and inefficiencies.

This project provides a **centralized, secure, and user-friendly platform** to manage employee appraisals efficiently.

The system enables:
- Employees to submit self-appraisals
- Managers to review and evaluate performance
- HR/Admin to manage appraisal cycles and reports

---

# ✨ Features

## 👨‍💼 Employee Features
- Secure Login and Authentication
- View Active Appraisal Cycles
- Submit Self-Appraisal Forms
- Add Achievements, Goals, Skills, and Improvements
- Save Draft and Edit Before Submission
- Track Appraisal Status and Progress
- View Manager Ratings and Feedback
- Access Previous Appraisal History
- Update Profile Information
- Receive Notifications and Deadline Alerts

---

## 👨‍💻 Manager Features
- Secure Login
- Switch Between Manager Mode and Employee Mode
- Submit Own Appraisal as Employee
- Assign Tasks to Employees
- Review Employee Self-Evaluations
- Assign Ratings and Feedback
- Track Team Appraisal Completion
- Monitor Employee Performance
- Generate Employee Reports
- Submit Final Recommendations

---

## 👩‍💼 HR / Admin Features
- Secure Login
- Create and Manage Appraisal Cycles
- Add, Edit, Delete Employees, Managers, Departments
- Assign Employees to Managers
- View Appraisal Forms
- Give Final Ratings and Promotion Decisions
- Manage Roles and Permissions
- Generate Reports
- Maintain Security and Data Integrity

---

## 📈 Dashboard Features
- Appraisal Statistics
- Employee Performance Metrics
- Progress Tracking
- Notifications and Alerts
- Report Generation
- Analytics Dashboard
